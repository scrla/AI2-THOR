# 모바일 로봇 인공지능 기술 개발
## 다중 로봇 작업 수행 테스트
---
| 순서 | 단계 | 주요 역할 및 설명 | 키워드 |
| :---: | :--- | :--- | :--- |
| **1.** | **명령 입력** | 사용자 또는 상위 AI로부터 고수준의 복합 임무를 자연어 또는 구조화된 형태로 입력받습니다. |
| **2.** | **작업 분해 (Task Decomposition)** | 입력된 복합 임무를 로봇들이 처리할 수 있는 논리적이고 실행 가능한 하위 작업들로 분할합니다. |
| **3.** | **병렬 작업 할당 (Parallel Assignment)** | 분해된 하위 작업을 각 로봇의 능력과 현재 chain 상태를 고려하여 병렬로 실행할 수 있도록 최적화하여 할당합니다. |
| **4.** | **로봇 이동 수행 (Execution)** | 할당된 작업을 각 로봇이 AI2-THOR 환경 내에서 실제 이동, 회피하여 명령을 완수합니다. |
